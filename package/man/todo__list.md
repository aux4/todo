# todo list

List all todo lists with progress.

## Usage

```bash
aux4 todo list [--file <path>]
```

## Variables

| Name | Description | Default |
|------|-------------|---------|
| `--file` | Todo file path | `.todo.json` |

## Example

```bash
aux4 todo list
```

```
Todo Lists:
  Work (1/3)
  Personal (0/2)
```
