# todo delete

Delete an entire todo list.

## Usage

```bash
aux4 todo delete <name> [--file <path>]
```

## Variables

| Name | Description | Default |
|------|-------------|---------|
| `name` | Todo list name (required) | |
| `--file` | Todo file path | `.todo.json` |

## Example

```bash
aux4 todo delete "Work"
```

```
Todo 'Work' removed.
```
